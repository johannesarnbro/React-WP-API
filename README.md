# Wordpress and React boilerplate with Babel and Webpack

## Dependencies
- iojs-v2.3.0
- Running Wordpress with WP-API (http://wp-api.org/)

## Install
```
nvm use iojs-v2.3.0
npm install
npm run dev
open http://localhost:3000
```

## Test
```
npm run test
```

## Config
- `/sourcecode/config.js`
